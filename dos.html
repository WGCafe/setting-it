WHAT IS DOC NETWORK?

Dos Network is a layer 2 chain-agnostic Decentralized Oracle Service network,providing real-time data oracle and computation oracle solution to mainstream blockchains.

DOS NETWORK FEATURES

  • DECENTRALIZE & REALIABLE

    A trustless solution without single point of failure.

  • REAL-TIME VERIFIABLE

    Near real-time request fullfinllment and on-chain verifiable response.

  • COMPATIBLE TO HETEROGENEOUS BLOCKCHAINS

    Support multiple chains and enable cross-chain interactions.

  • VRF ADVANCED CRYPTOGRAPH

    Driven by Random Function and advanced cryptographic signatures

  • NON-INTERACTIVE VERIFIABLE COMPUTATION

    Unbounded computation power for smart contracts secured by SNARK-based computation oracle

  • CRYPTO INCENTIVIZED

    Dynamic network incentivized by honest participation reward
